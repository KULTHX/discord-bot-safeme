# 🤖 KULTHX SAFEME Discord Bot مع واجهة ويب

بوت ديسكورد متقدم لحماية نصوص Roblox مع **واجهة ويب سهلة الاستخدام** لإدخال توكن البوت بدون الحاجة لتعديل الملفات.

## ✨ المميزات الجديدة

### 🌐 واجهة ويب تفاعلية
- **إدخال آمن للتوكن**: لا حاجة لملفات .env أو متغيرات البيئة
- **تشغيل فوري**: البوت يبدأ العمل فور إدخال التوكن
- **مراقبة الحالة**: عرض حالة البوت في الوقت الفعلي
- **تصميم عربي**: واجهة مصممة خصيصاً للمستخدمين العرب

### 🔒 حماية متقدمة للنصوص
- تشفير AES-256-CBC للنصوص
- روابط آمنة يمكن الوصول إليها من Roblox فقط
- نظام معرفات فريدة لكل نص
- حماية من الوصول غير المصرح به

### 🤖 أوامر ديسكورد تفاعلية
- `/حماية` - حماية نص Roblox وإنشاء رابط آمن
- `/نصوصي` - عرض جميع النصوص المحمية
- `/حذف` - حذف نص محمي
- `/إحصائيات` - عرض إحصائيات البوت
- `/مساعدة` - دليل الاستخدام

## 🚀 البدء السريع (الطريقة الجديدة)

### 1️⃣ تثبيت المكتبات
```bash
npm install
```

### 2️⃣ تشغيل الواجهة الويب
```bash
npm start
# أو
npm run web
```

### 3️⃣ فتح المتصفح
اذهب إلى: `http://localhost:3000`

### 4️⃣ إدخال التوكن
1. احصل على توكن البوت من [Discord Developer Portal](https://discord.com/developers/applications)
2. الصق التوكن في الحقل
3. اضغط "تشغيل البوت"

**🎉 هذا كل شيء! البوت سيعمل فوراً بدون أي تعقيدات!**

## 🎮 استخدام النصوص في Roblox

بعد حماية النص، ستحصل على رابط مثل:
```
http://localhost:3000/script.lua?id=abc123def456
```

استخدمه في Roblox:
```lua
loadstring(game:HttpGet("http://localhost:3000/script.lua?id=abc123def456"))()
```

## 🛡️ الأمان والحماية

### حماية التوكن
- التوكن لا يُحفظ في أي ملف
- يبقى في ذاكرة الخادم فقط
- يُحذف عند إيقاف الخادم

### حماية النصوص
- تشفير AES-256-CBC
- مفاتيح تشفير قابلة للتخصيص
- وصول محدود من Roblox فقط

### Rate Limiting
- 100 طلب كل 15 دقيقة لكل IP
- حماية من هجمات DDoS
- تسجيل محاولات الإساءة

## 🔧 إعدادات متقدمة

### متغيرات البيئة الاختيارية
```env
PORT=3000                    # منفذ الخادم
HOST=0.0.0.0                # عنوان الاستماع
BASE_URL=http://localhost:3000  # الرابط الأساسي للنصوص
ENCRYPTION_KEY=your-secret-key  # مفتاح التشفير
RATE_LIMIT_WINDOW_MS=900000     # نافزة Rate Limiting
RATE_LIMIT_MAX_REQUESTS=100     # الحد الأقصى للطلبات
```

### تشغيل في الإنتاج
```bash
# استخدام PM2
pm2 start web-server.js --name "kulthx-safeme-web"

# أو استخدام nohup
nohup npm start > server.log 2>&1 &
```

## 📊 API Endpoints

### `GET /`
الصفحة الرئيسية لواجهة الويب

### `POST /api/bot/start`
تشغيل البوت بتوكن معين
```json
{
  "token": "your_bot_token_here"
}
```

### `GET /api/bot/status`
الحصول على حالة البوت الحالية

### `GET /script.lua?id=<script_id>`
تنفيذ النص المحمي (للاستخدام من Roblox فقط)

## 🛠️ استكشاف الأخطاء

### ❌ "فشل في تشغيل البوت: An invalid token was provided"
**الحل:** تأكد من صحة توكن البوت من Discord Developer Portal

### ❌ "خطأ في الاتصال بالخادم"
**الحل:** تأكد من تشغيل الخادم على المنفذ الصحيح

### ❌ "البوت لا يستجيب للأوامر"
**الحل:** تأكد من أن البوت لديه الصلاحيات المطلوبة في الخادم
